# umi project

## Getting Started

Install dependencies,

```bash
$ yarn
```

Start the dev server,

```bash
$ yarn start
```
## 本地运行

- 本地运行（启动 mock）`npm start`
- 本地运行（禁用 mock）`npm run dev-no-mock`
